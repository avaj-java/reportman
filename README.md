# reportman
Java util for Excel


## build.gradle 
1. Set repositories
    ```gradle
    allprojects {
      repositories {
        ...
        maven { url 'https://jitpack.io' }
      }
    }
    ```
2. Set dependencies
    ```gradle
    dependencies {
      implementation 'com.github.avaj-java:reportman:0.3.2'
    }
    ```
